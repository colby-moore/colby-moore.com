Fantasy Esports

Fantas-e is an ongoing project currently being re-built using Vue.js. Fantas-e is an idea for bringing fantasy sports drafts to the world of Esports. A new player draft for each tournament; 10 players consisting of 2 Awpers, 2 Entries, 2 HS, 2 Pistol, 2 Fragers, and will feature a custom point system.

Jeera

Built with Vue.js and using firebase, Jeera is a SPA using vue-router, and is a re-make of the popularly used Atlassian product Jira for project and issue tracking. Jeera contains a homepage where you can see all tasks, and a kanban board where you can add and remove tasks.

Compiler

A compiler built with javascript that supports a toy language found here. The source code input goes through the processes of Syntax Analysis, Semantic Analysis, Intermediate Code Generation, Code Generation, and mantains a Symbol Table through compilation.

Fullpage

Fullpage is a website template built using fullpage.js and particle.js. It is a fullpage scroll controlled site with horizontal and vertical scrolling slides and sidenav icon bar.
